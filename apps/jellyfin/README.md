# Jellyfin client

Controller-friendly Jellyfin media client for Steam Big Picture on a living-room PC.

## Status

Scaffold only. Stack, features, and UI direction are not locked yet.

## Planned shape

- Connect to a self-hosted Jellyfin server (URL + user auth)
- Browse libraries and play media
- Full gamepad navigation suitable for Steam Big Picture
- Packaged so it can be added as a non-Steam game on Windows

## Local notes

Once the app is implemented, this README will cover:

- How to configure the Jellyfin server URL
- How to run in development
- How to build a Windows executable for Steam

## Steam Big Picture checklist (target)

- [ ] Launches fullscreen / borderless
- [ ] Controllable without mouse/keyboard
- [ ] Focus order works with D-pad / stick
- [ ] Back / B exits menus cleanly
- [ ] Readable from couch distance
